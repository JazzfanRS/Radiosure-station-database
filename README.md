# Radiosure-station-database 

A place to share the RadioSure station database since the website and forums are gone. You can find discussions at https://www.reddit.com/r/RADIOSURE/

If you use this database and modify it for your own applications other then RadioSure, please label them as such, and keep this root for RadioSure only.
I am new to github and only use the RadioSure application, so my apologies if this is cumbersome to use, as interested parties we are looking at the best way to keeep the database active and accessible to all volunteers to modify.
